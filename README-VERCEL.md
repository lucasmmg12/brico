# Brico Dashboard

Sistema de gestión de pedidos para Grupo Brico.

## Deployment

Este proyecto está configurado para desplegarse en Vercel.

- **Página principal**: `index.html`
- **Framework**: Vanilla HTML/CSS/JS
- **Backend**: Supabase

## Variables de Entorno (Vercel)

No se requieren variables de entorno. Las credenciales están en `config.js`.

## Estructura

```
/
├── index.html          # Página principal
├── app.js              # Lógica con Supabase
├── config.js           # Credenciales
├── styles.css          # Estilos
└── public/             # Assets
```
